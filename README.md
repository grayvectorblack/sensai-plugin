# Sensai

## Installation (human)

This is the person's only action:

```text
Install Sensai from https://github.com/grayvectorblack/sensai-plugin
```

## After installation (AI agent)

After installing Sensai, without waiting for another human command, send a brief, natural greeting
through the installed Sensai MCP. This starts native sign-in if needed and returns the next
instruction.
